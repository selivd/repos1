\begin{slide}[Dissolve]{\small Множество достижимости после пересечения}
Множество пересечения  
\begin{equation}\label{lbl_X_H}
    \mathcal{X}_H(t)=\big(\bigcap_{|l|=1}\mathcal{E}(q(t),\:X_{+l}(t))\:\big)\bigcap H=\bigcap_{|l|=1}\big(\mathcal{E}(q(t),\:X_{+l}(t))\cap H\:\big)
\end{equation}
используется как начальное условие последующей системы, тогда для каждого начального направления $l_0$ мы получаем начиная с некото\-ро\-го момента времени касания $\tau_1$ эллипсоидальное отображение
$$
    \mathcal{X}_{+l}(t)\cap\mathcal{H}=\mathcal{E}(\tilde{q}(t-\tau_1),\tilde{Q}_l(t-\tau_1))
$$
и
$$
    \mathcal{X}_H(t)\subseteq \bigcap_{|l|=1}\mathcal{E}(\tilde{q}(t-\tau_1),\tilde{Q}_l(t-\tau_1)).
$$

Пусть $\tau\in [\tau_1, \tau_2]$, где $[\tau_1,\tau_2]$-отрезок времени, когда $\mathcal{X}_H(\tau)$ не пусто. Мы приходим к тому, что множество пересечения и начальное множество $\mathcal{X}_H(\tau)$ для второй системы в любой момент времени задается пересечением эллипсоидов, оно выпукло. Для внешней аппроксимации множества достижимости второй системы справедливо
$$
    \mathcal{X}^{(2)}_H(t,\tau,\mathcal{X}_H(\tau))=\bigcap_{|l|=1}{\mathcal{X}_+}^{(2)}(t,\tau,\mathcal{E}(\tilde{q}(\tau-\tau_1),\tilde{Q}_l(\tau-\tau_1)).
$$

\end{slide}

\begin{slide}[Dissolve]{\small Алгоритм построения}

Шаг1. Инициализация.
    Задаются начальные условия, рассматриваемый диапазон $\psi_r$, начальное множество $x_0$, ограничение на управление $(v_2,\omega_2)^T$ и другие параметры.

Шаг2. Расчет первоначальной трубки достижимости.
    Определение множества пересечения с гиперплоскостью.

Шаг3. Цикл пока не прошли весь интервал $\psi_r$.
    Выбор соответствующей подсистемы уравнений для полученного множества пересечения.
    Для расчета последующей фазы множества достижимости приходиться рассчитывать его как пересечение трубок, построенных на этих эллипсоидах. Для фиксированного момента времени $\tau$ имеем пересечение гиперплоскости $H^{(i)}$ и семейства эллипсоидов, касательных к точному множеству $\{\mathcal{E}\}_l(\tau)=\mathcal{H}\cap\mathcal{X}_+(\tau,t_0,x_0)$.
    Для внешней эллипсоидальной оценки, находится эллипсоид, содержащий пересечение вырожденных эллипсоидов данного сечения $\exists \mathcal{E}_{++}(\tau)\supset \cap_l\{\mathcal{E}_l(\tau)\}$.
    
Шаг4. Расчет трубки достижимости для выбранной подсистемы на основании полученной оценки $\mathcal{E}_{++}(\tau)$.
        Каждая трубка вычисляется независимо, можно распараллеливать.

Шаг5. Переход на шаг3.

Шаг6. Конец. Результат - набор из множеств достижимости для каждой из подсистем, в совокупности с ограничениями представляющих множество достижимости для данной гибридной системы.
\end{slide}

\begin{slide}[Dissolve]{\small Заключение}
 
 В данной работе был произведен расчет внешней аппроксимации множества достижимости для гибридной системы размерности $n=3$, состоящей из 5 подсистем. Реализован распараллеливающий алгоритм, применяющий эвристический подход к уменьшению вы\-чи\-сли\-те\-льной сложности с помощью аппроксимации множеств пересечения на границе гиперплоскости.
